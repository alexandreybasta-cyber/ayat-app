# Versets pour Votre Âme - Projet Terminé

## 🎉 Site Web Déployé avec Succès

Votre site web "Versets pour Votre Âme" a été créé et déployé avec succès !

**URL du site :** https://ntohvnen.manus.space

## 📋 Fonctionnalités Implémentées

### ✅ Sélection d'Humeurs
Le site propose 9 humeurs différentes :
- 😢 Tristesse / Chagrin
- 😰 Anxiété / Peur
- 😡 Colère / Frustration
- 💪 Besoin de motivation / Découragement
- 🕊️ Recherche de paix / Sérénité
- 😊 Gratitude / Joie
- 🤲 Besoin de pardon / Repentir
- 🧭 Recherche de guidance / Confusion
- 🌅 Espoir / Optimisme

### ✅ Affichage des Versets
Pour chaque humeur, 10 versets coraniques sont présentés avec :
- Texte arabe original (avec police Amiri pour une belle présentation)
- Traduction française
- Référence (sourate et numéro de verset)
- Contexte explicatif

### ✅ Système de Favoris
- Possibilité d'ajouter/supprimer des versets en favoris
- Sauvegarde automatique dans le navigateur (localStorage)
- Page dédiée aux favoris
- Compteur de favoris visible sur la page d'accueil

### ✅ Design Moderne et Responsive
- Interface élégante avec dégradés de couleurs apaisants
- Design responsive (fonctionne sur mobile et desktop)
- Animations douces et transitions fluides
- Typographie optimisée pour le texte arabe

## 🛠️ Technologies Utilisées

- **Frontend :** React avec Vite
- **Styling :** Tailwind CSS + CSS personnalisé
- **Composants UI :** shadcn/ui
- **Icônes :** Lucide React
- **Polices :** Amiri et Noto Sans Arabic pour le texte arabe
- **Déploiement :** Manus Platform

## 📊 Données Incluses

Le site contient actuellement 90 versets coraniques (10 par humeur) avec :
- Texte arabe authentique
- Traductions françaises précises
- Références complètes des sourates
- Contextes explicatifs pour chaque verset

## 📱 Comment Utiliser le Site

1. **Visitez** https://ntohvnen.manus.space
2. **Choisissez** votre humeur actuelle parmi les 9 options
3. **Lisez** les 10 versets sélectionnés pour cette humeur
4. **Ajoutez** vos versets préférés aux favoris en cliquant sur le cœur
5. **Consultez** vos favoris via le bouton "Mes Favoris"

Le site est maintenant entièrement fonctionnel et accessible à tous !
